# Kittygram Final

## Описание проекта
**Kittygram** — это социальная сеть для любителей котиков, позволяющая пользователям делиться фотографиями своих питомцев, добавлять их характеристики (порода, цвет, возраст) и просматривать публикации других участников. 

### Цели проекта:
* Организация полноценной контейнеризации монолитного приложения для локальной разработки и продакшена.
* Настройка веб-сервера Nginx в качестве единой точки входа (Gateway) для маршрутизации запросов и безопасной раздачи статики/медиа.
* Реализация автоматического CI/CD пайплайна с использованием GitHub Actions для тестирования кода и сборки Docker-образов на Docker Hub.

### Основные функции:
* Регистрация, аутентификация пользователей и управление профилями (на базе Djoser).
* Добавление, редактирование и удаление карточек котиков с возможностью загрузки изображений.
* Просмотр ленты питомцев с фильтрацией по породам.
* Полноценная административная панель Django для управления контентом.

---

## Стек технологий
* **Backend:** Python 3.9, Django 3.2, Django REST Framework, Gunicorn, Djoser
* **Database:** PostgreSQL 13
* **Frontend:** React, HTML, CSS
* **Infrastructure:** Docker, Docker Compose, Nginx (Gateway)
* **CI/CD:** GitHub Actions, Flake8, Docker Hub
